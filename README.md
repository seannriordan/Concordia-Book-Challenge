# README

Concordia Publishing House Coding Challenge

Simple Book Catalog.
Books can have a Title, Author, ISBN, and Published Date. (Title, Author, and ISBN are required fields. Published Date is not.) You can add, edit, and delete the books. And You can sort by any of the headers in ascending or descending order by clicking on them.

* Ruby version: 3.1.2
* Rails version: 7.0.3.1
